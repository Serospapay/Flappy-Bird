# Flappy Bird - Enhanced Edition

Покращена версія класичної гри Flappy Bird, реалізована на Python з використанням pygame. Проект розроблений для кваліфікаційної роботи бакалавра.

## Опис

Гравець керує птахом, який має пролітати через перешкоди (труби), не зіштовхуючись з ними. За кожну пройдену трубу нараховується очко. Гра включає багато покращень та додаткових функцій.

## Основні функції

### 🎮 Геймплей
- **Класичний геймплей Flappy Bird** з покращеннями
- **Режими**: Звичайна гра, Щоденний виклик (фіксований seed), Скіни
- **Різні типи перешкод**: звичайні, рухомі, зменшувальні, з шипами, роздвоєні
- **Power-ups**: щит, повільний час, подвійні очки, монети, ghost (прохід крізь труби)
- **Динамічна складність**: гра автоматично ускладнюється з ростом рахунку
- **Система монет**: збирайте монети для досягнень

### 🎨 Візуальні ефекти
- **Паралакс фон**: багатошаровий фон з ефектом глибини
- **Система частинок**: візуальні ефекти при зіткненнях та зборі бонусів
- **Анімації**: плавна ротація птаха та рух об'єктів
- **Різнокольорові труби**: візуальна відмінність типів перешкод

### 🔊 Звук та музика
- **Звукові ефекти**: стрибок, колізія, збір монет, досягнення
- **Налаштування гучності**: окремо для музики та звукових ефектів
- **Програмна генерація звуків**: не потребує зовнішніх файлів

### 📊 Системи та статистика
- **Система досягнень**: 10 різних досягнень для розблокування
- **Статистика**: відстеження найкращого рахунку, середнього рахунку, загального часу гри
- **Збереження прогресу**: автоматичне збереження в JSON файл
- **Меню налаштувань**: налаштування гучності, складності, теми (світла/темна)
- **Скіни птаха**: розблокування за монети

## Встановлення

### Автоматичне встановлення (Windows)
1. Запустіть `install.bat` - встановить всі необхідні залежності автоматично

### Ручне встановлення
1. Встановіть залежності:
```bash
pip install -r requirements.txt
```

## Запуск

### Windows (Рекомендовано)
1. **Через Провідник**: Двічі клацніть на `run.bat` - гра запуститься автоматично
2. **Через Command Prompt (CMD)**:
```bash
run.bat
```
3. **Через PowerShell**:
```powershell
.\run.bat
```

### Інші платформи / Ручний запуск
```bash
python src/game.py
```

## Керування

### В меню
- **↑/↓** - навігація по меню
- **SPACE/ENTER** - вибір опції
- **ESC** - вихід

### Під час гри
- **SPACE** або **CLICK** - стрибок птаха
- **DOWN** - dash (ривок вгору, cooldown 2 сек)
- **ESC** - пауза / повернення в меню

### Після Game Over
- **SPACE/ENTER** - рестарт гри
- **ESC** - повернення в меню

## Особливості реалізації

### Продуктивність
- Оптимізований ігровий цикл (60 FPS)
- Ефективна система колізій
- Об'єктно-орієнтована архітектура для легкого розширення

### Архітектура
- **Компонентний підхід**: кожна система в окремому модулі
- **State Pattern**: різні стани гри (меню, гра, game over, налаштування)
- **Observer Pattern**: система досягнень та подій
- **Singleton Pattern**: менеджери систем (звук, збереження)

### Функціональність
- Автоматичне збереження прогресу
- Динамічна складність з прогресією
- Різні типи перешкод для різноманітності геймплею
- Система досягнень для мотивації гравця

## Технічні деталі

- **Мова**: Python 3.14+
- **Бібліотека**: pygame-ce 2.5.6+
- **Формат збереження**: JSON
- **Архітектура**: Об'єктно-орієнтована з модульною структурою
